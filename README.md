# SMARTDRIVE-RENTALS
This project shows my understanding of JAVA GUI, OOP and JDBC

## Database Setup

This project uses MySQL and JDBC for database connectivity.

To run the project:
1. Install MySQL Server
2. Create a database named `smartdrive_db`
3. Create the `cars` table as provided
4. Download MySQL Connector/J
5. Add the connector JAR to the project classpath

The JDBC driver is not included in this repository.
